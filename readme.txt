== Description ==

Convenient and user-friendly online currency converter widget. Easily implemented to your website with minimum efforts. And 100% free. 
You can choose from 190+ world currencies with an extended cryptocurrency list. Pick one among 7 available languages.  

The currency converter widget/calculator powered by [mconvert.net](https://mconvert.net)

Set up currency converter table and do calculations whenever you need it without going to different websites. 

All the data is coming from a 3rd party server (mconvert.net) so it doesn’t create an overload for your page and also doesn't store or save any data. Don’t hesitate and give it a chance.

GDPR: This widget does not store any user data at mconvert.net servers

== Installation ==

1. Install the "MConvert Currency converter" plugin
2. Activate plugin in "Plugins page".
3. Open Widgets page in admin panel, add it to website
4. Configure plugin as you want.
